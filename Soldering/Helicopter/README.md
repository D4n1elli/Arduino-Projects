# Toy Helicopter Project 🚁
This project was made on 18/04/2026 and committed 1 day later

## How I made it

1. First, used silicon glue and wood to make the helicopter body
2. Made the propeller separately
3. Soldered the push-button to the DC motor wires
4. Used instant glue to attach the motor to the helicopter (silicon glue wouldn't work for it since it doesn't glue plastic)
5. Glued the propeller to the DC motor

## How it works
You can interact with the helicopter by pressing the push-button and the propeller spins like a real one.
The push-button allows the current to pass when pressed, closing the circuit.

## Materials used
- Wood sticks
- Silicon glue and instant glue
- DC motor
- Wires
- Push-button
- Arduino as battery
