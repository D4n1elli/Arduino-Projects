# Buzzer With Push-Button

## Goal
I made this simple circuit with the intent to test the new soldering set I bought, since my old one was not working properly and made my soldering messy and weak. This time, it clearly got better and stronger, as I used soldering flux for the first time.

## Circuit
### How it works
The current leaves the positive side of the battery, traveling through the wire and reaches the button. If the button is pressed, the current travels to the buzzer and goes back to the battery at the negative side, closing the circuit.

### How I made it
First, i soldered the buzzer into the prototype board and the button right next to it, soldered the wires (paperclip, in this project) and folded them so they would touch the battery. The battery body is not soldered so it is easy to be replaced or removed to be used in another project.

## Components
- 1 prototype board
- 1 buzzer
- 1 push-button
- 3 Wires (I used paperclips in this project)
